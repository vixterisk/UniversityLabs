«Извлекатель корня» – утилита, позволяющая найти квадратный корень из числа (Квадратным корнем из числа x называют такое число, квадрат которого будет равен x). На главной странице сайта необходимо ввести число, из которого вы желаете извлечь корень, по желанию указав значение точности результата, и нажать на кнопку «Вычислить», после чего ответ будет выдан в соответствующем поле.
«Извлекатель корня» поддерживает:
  1. Арифметические (Арифметическим квадратным корнем из числа а называют неотрицательное число, квадрат которого равен а);
  2. Корень из нуля;
  3. Комплексные (комплексные числа - числа вида a+bi, где a, b – вещественные числа, I – мнимая единица);
  4. Корень из «Длинного» числа (Длинное число – число, разрядность которого превышает длину машинного слова данной вычислительной машины – в современных компьютерах числа разрядностью более 32 или 64 знаков);
  5. Аналитические
Также возможно задать точность найденного корня, указав в соответствующем поле желаемое число знаков после запятой.

"Root extractor" is a utility that allows you to find the square root of a number (The square root of a number x is a number whose square will be equal to x). On the main page of the site, you must enter the number from which you want to extract the root, optionally indicating the value of the accuracy of the result, and click on the "Calculate" button, after which the answer will be given in the appropriate field.
Root extractor supports:
  1. Arithmetic (The arithmetic square root of the number a is a non-negative number whose square is equal to a);
  2. Root from zero;
  3. Complex (complex numbers are numbers of the form a + bi, where a, b are real numbers, I is an imaginary unit);
  4. Root of a "Long" number (Long number is a number whose bit depth exceeds the length of a machine word of a given computer - in modern computers, numbers are more than 32 or 64 characters long);
  5. Analytical.
It is also possible to set the precision of the found root by specifying the desired number of decimal places in the corresponding field.

"Root extractor" es una utilidad que te permite encontrar la raíz cuadrada de un número (La raíz cuadrada de un el número x es un número cuyo cuadrado será igual ax). En la página principal del sitio, debe ingresar el número del que desea extraer la raíz, indicando opcionalmente el valor de la precisión de la resultado y haga clic en el botón "Calcular", después de lo cual se dará la respuesta en el campo.
El extractor de raíces admite:
  1. Aritmética (La raíz cuadrada aritmética del número a es un número no negativo cuyo cuadrado es igual a a);
  2. Raíz desde cero;
  3. Complejo (los números complejos son números de la forma a + bi, donde a, b son números reales, I es un unidad imaginaria);
  4. Raíz de un número "largo" (el número largo es un número cuya profundidad de bits excede la longitud de un palabra de máquina de una computadora determinada: en las computadoras modernas, los números son más de 32 o 64 caracteres de largo);
  5. Analítico.
También es posible establecer la precisión de la raíz encontrada especificando el número deseado de posiciones decimales en el campo correspondiente.
